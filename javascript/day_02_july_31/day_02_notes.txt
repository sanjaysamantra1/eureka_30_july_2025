How to use CSS in HTML file
===========================
1. inline css (style attribute) (restricted to an element)
    <div style='color:red'>This is a div</div>
2. internal css (style tag) (restricted to 1 HTML file)
   <style></style>
3. external css
    a. write css code in a separate .css file
    b. include that css file in HTML file using <link>

Types of javascript
====================
1. Internal Javascript <script></script>
2. External javascript (separate .js file)


How to write and execute Javascript code
========================================= 
1. Browser (FrontEnd Development)       
   a. create a javascript file with some javascript content (abc.js)
   b. include that javascript file into HTML file 
      <script src='abc.js'></script>
   c. open HTML file in browser (alt+b)

2. NodeJS (backEnd Development)
    a. create a javascript file with some javascript content (abc.js)
    b. run the javascript file with the below command
       node <filename>.js  (make sure nodejs is installed)

Note : Every javascript code that runs on browser might not run on nodejs , and vice-versa

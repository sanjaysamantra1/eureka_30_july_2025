Web Development
===============
-Developing a website

1. FrontEnd : User Interface ( Look & Feel )
        HTML : Content ( paragraphs, images, list , table , form , audio, video )
        CSS :  Styling ( border , background , width , height , margin , padding )
        Bootstrap : CSS Library ( Pre-defined css classes , responsiveness )
        Javascript : Behaviour 
        Angular/React : Javascript frameworks (RAD , CLI , Template system, Routing , Form handling )
2. BackEnd : Develop REST Apis ( Java / .net / Python / PHP / Nodejs / Go)
3. Database : Holds the data ( oracle , mySQL , MongoDB , Postgree)
